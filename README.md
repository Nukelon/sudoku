# Sudoku

一个纯前端的数独小游戏，可直接部署在 GitHub Pages 或任意静态服务器上。支持触控与键盘输入，提供多种难度、提示与冲突高亮。

## 支持的变体
- 经典 9×9 数独
- 迷你数独：4×4、6×6（2×3 宫）、7×7 不规则宫
- 超数独（Hyper Sudoku / Windoku）
- 对角线数独（Sudoku X）
- 环形数独（以环与射线为区域，采用同心环视觉布局）
- 杀手数独（带笼子求和与不重复约束，内置生成器带超时保护）

## 使用方法
1. 打开 `index.html` 即可开始游戏，无需构建步骤。
2. 通过右上角选择难度并点击“新游戏”生成棋盘。
3. 选中格子后可用数字键或下方数字面板输入，按 Backspace/Delete/空格清除。
4. 点击“提示”填入一个随机空格；点击“检查”验证当前答案。

## 部署到 GitHub Pages
1. 将代码推送到仓库的 `main` 或 `gh-pages` 分支。
2. 在仓库设置中开启 Pages（来源选择对应分支 / 根目录）。
3. 稍等几分钟后即可通过 Pages 域名访问游戏。
